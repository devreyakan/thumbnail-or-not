Thumbnail or Not plugin adds a "Thumbnail Status" column to the media library to check if an image is used as a thumbnail.

== Description ==

The Thumbnail or Not plugin enhances the WordPress media library by adding a new column, "Thumbnail Status," which shows whether an image is used as a post thumbnail (featured image). It helps site administrators to easily identify which images are linked to posts as thumbnails and which are not.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/thumbnail-or-not` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'Media' section to view the new "Thumbnail Status" column.

== Frequently Asked Questions ==

= How does this plugin work? =

Once activated, the plugin adds a new column in the WordPress media library labeled "Thumbnail Status." The column will display "Yes" if the image is used as a post thumbnail, and "No" if it is not.

= Can I sort the media items by Thumbnail Status? =

Yes, you can click on the "Thumbnail Status" column header to sort images based on whether or not they are used as post thumbnails.
